# HungerFuel 🍽️

## Overview

HungerFuel is a restaurant review and rating application that helps users discover restaurants, explore menus, and share their dining experiences. The platform enables users to rate restaurants, write reviews, and view feedback from other customers, making it easier to choose the best place to eat.

---

## Features

- User Registration and Login
- Browse Restaurants
- View Restaurant Menus
- Search Restaurants
- Submit Ratings (1–5 Stars)
- Write and Edit Reviews
- View Customer Reviews
- Average Rating Calculation
- Restaurant Details Page
- Responsive User Interface
